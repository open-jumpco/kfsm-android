= KFSM Android Demo

A simple application to demonstrate implementing link:https://github.com/open-jumpco/kfsm[KFSM] with the classic Turnstile FSM.


== Generated State Table

=== TurnstileFSM State Map

|===
| Start | Event[Guard] | Target | Action

| LOCKED
| COIN
| UNLOCKED
|  `{unlock()}`

| UNLOCKED
| PASS
| LOCKED
|  `{lock()}`

| UNLOCKED
| COIN
| UNLOCKED
|  `{returnCoin()}`
|===
